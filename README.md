# sim1000G


R package for simulating genetic marker data. 

#### Author: Apostolos Dimitromanolakis
#### Maintainer: Apostolos Dimitromanolakis

## Installation

The most stable version can be found at CRAN:

```R
install.packages("sim1000G")
```
